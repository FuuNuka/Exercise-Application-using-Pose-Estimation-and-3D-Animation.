# 🏋️‍♂️ Exercise Application using Pose Estimation and 3D Animation

[Slie And Use Case] (https://canva.link/fksqqmziqr1ml8o)

แอปพลิเคชันสำหรับการออกกำลังกายแบบ Interactive 3D ที่สามารถตรวจจับการเคลื่อนไหวของผู้ใช้งานได้แบบ Real-time (Pose Estimation) โดยใช้กล้องเว็บแคม และนำข้อมูลการเคลื่อนไหวไปควบคุมตัวละคร 3D (3D Avatar) ภายในเกมแบบอัตโนมัติ โปรเจกต์นี้เป็นส่วนหนึ่งของปริญญานิพนธ์ (Thesis)

## 🛠 Tech Stack & Tools
- **Game Engine:** Unity 3D
- **Programming Language:** C#
- **AI & Computer Vision:** Google MediaPipe (Pose Estimation)

## ✨ Key Features (ความสามารถหลัก)
- **Real-Time Motion Tracking:** ตรวจจับข้อต่อและสรีระร่างกายของผู้ใช้ผ่านกล้อง Web Camera 
- **3D Avatar Mapping:** แปลงจุดพิกัด (Landmarks) จากร่างกายมนุษย์ ไปสั่งการการเคลื่อนไหวของตัวละคร 3D
- **Interactive UI:** ส่วนติดต่อผู้ใช้งานที่เข้าใจง่าย ออกแบบมาสำหรับการออกกำลังกาย

## 🚀 How to Run (วิธีการติดตั้งและทดสอบ)
1. ติดตั้ง **Unity Version [ใส่เวอร์ชันที่คุณใช้]**
2. โคลนโปรเจกต์นี้ลงเครื่อง
3. เปิดโปรเจกต์ผ่าน Unity Hub
4. โปรเจกต์นี้จำเป็นต้องเชื่อมต่อ **Webcam** ในการรัน
5. เปิด Scene หลัก แล้วกด Play เพื่อทดสอบ

## 🎓 About This Project
โปรเจกต์นี้จัดทำขึ้นเพื่อการศึกษาและการวิจัยในหัวข้อปริญญานิพนธ์ โดยมุ่งเน้นที่การนำเทคโนโลยี AI ด้าน Computer Vision มาประยุกต์ใช้กับ Game Engine เพื่อสร้างประสบการณ์การออกกำลังกายรูปแบบใหม่
